CSSとは？

HTMLの見た目を装飾する言語。色・文字サイズ・配置などを制御します。

セレクタの種類

  • タグセレクタ：p { color: red; }
  • クラスセレクタ：.classname { ... }
  • IDセレクタ：#idname { ... }
  • 子孫セレクタ：div p { ... }

代表的なプロパティ

  • color：文字色
  • background-color：背景色
  • font-size：文字サイズ
  • margin / padding：外側/内側の余白
  • border：枠線
  • display：block / inline / flex / grid

サンプルコード

p {
  color: #0984e3;
  font-size: 18px;
  margin-bottom: 15px;
}

.container {
  display: flex;
  flex-direction: column;
  gap: 20px;
}
