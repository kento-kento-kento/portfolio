HTMLとは？

Webページの構造を作るための言語です。

基本タグ

  • <h1>〜<h6> : 見出し
  • <p> : 段落
  • <a href="URL"> : リンク
  • <img src="画像URL"> : 画像
  • <ul>/<ol>/<li> : リスト
  • <div> : ブロック要素
  • <span> : インライン要素
  • <form> : フォーム

属性の例

  • href：リンク先
  • src：画像URL
  • alt：画像の代替テキスト
  • class / id：CSSやJSで指定する名前
  • rel：linkタグなどで関係を指定

サンプルコード


<!DOCTYPE html>
<html>
  <head>
    <title>サンプルページ</title>
  </head>
  <body>
    <h1>こんにちは</h1>
    <p>HTMLの基礎ページです。</p>
  </body>
</html>
